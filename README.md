### Code Review

- PR은 리더의 리뷰 후 머지합니다.
- 리뷰어가 요청한 변경 사항은 반드시 반영하거나 이유를 설명합니다.

다음 항목 위주로 리뷰합니다.  
```1.네이밍/가독성```, ```2.중복/불필요한 로직```, ```3.예외 처리```
